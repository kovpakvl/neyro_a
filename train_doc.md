## Назначение
Документ описывает работу скрипта `train.py`, который выполняет обучение нейросетевой модели регрессии углов взгляда, оценку качества и сохранение артефактов модели для дальнейшего использования в API.

## Входные данные
Скрипт использует CSV файл `out/gaze_dataset.csv`, сформированный после этапа подготовки данных. В наборе присутствуют 11 числовых признаков, целевые углы `H` и `V`, а также идентификатор пользователя `person_id` для корректной групповой валидации.

## Процесс обучения
Для каждого угла обучается отдельная модель MLPRegressor. Перед обучением выполняется стандартизация признаков с помощью StandardScaler. Подбор гиперпараметров проводится на небольшой сетке, ориентируясь на минимальную среднюю ошибку по групповой кросс валидации. Для оценки качества строятся графики средних абсолютных ошибок для обучающего и проверочного наборов, которые сохраняются в файл `out/models/quality_cv.png`.

## Результаты и артефакты
По завершении обучения сохраняются файлы `scaler.pkl`, `model_h.pkl` и `model_v.pkl` в каталоге `out/models`. Эти артефакты используются API сервисом без повторного обучения.

## Порядок запуска
Запуск выполняется командой `python train.py`. Скрипт проверяет наличие входного CSV и завершает работу, если файл не найден. Рекомендуется предварительно выполнить подготовку данных, чтобы сформировать `out/gaze_dataset.csv`.
