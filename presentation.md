# Презентация результатов (модули А, Б, В)

---

# Слайд 1. Цель и итог

- Построено решение для определения направления взгляда
- Выполнены три этапа: данные → модель → тестирование
- Подготовлены API и GUI для демонстрации

---

# Слайд 2. Модуль А — анализ и подготовка данных

- Распарсен датасет Columbia Gaze
- Реализованы инвариантные признаки в `features.py`
- Подготовка и очистка данных в `preprocess_dataset.py`
- Идея: получить устойчивые признаки, не зависящие от масштаба и сдвига

---

# Слайд 3. Модуль Б — модель и интерфейсы

- Нейросетевая регрессия углов взгляда (`train.py`)
- HTTP API для интеграции (`api.py`)
- GUI клиент для визуализации (`app_api_client_simple.py`)
- Идея: разделить обучение и инференс, упростить интеграцию

---

# Слайд 4. Модуль В — тестирование

- Модульные сценарии для ключевых функций
- Автоматизированный запуск: `run_module_v_tests.py`
- Отчёт и скриншот результатов в `out/module_v`
- Идея: быстрое подтверждение качества без тяжёлых зависимостей

---

# Слайд 5. Результаты и дальнейшие шаги

- Сформирована полная цепочка: данные → модель → API → тесты
- Модель готова к демонстрации и интеграции
- Возможные улучшения: расширение набора тестов, сбор реальных метрик
