## Назначение
Документ описывает работу скрипта `app_api_client_simple.py`, который реализует простое графическое приложение для демонстрации работы API определения направления взгляда. Приложение отображает видеопоток с камеры, текстовое направление взгляда, числовые углы и точку на координатной сетке.

## Требования и подготовка
Для работы требуется установленный Python, зависимости из `requirements.txt`, а также запущенный HTTP API сервис. Камера должна быть доступна системе, а адрес API должен быть корректным.

## Порядок запуска
Сначала необходимо запустить API сервис. Затем выполняется команда `python app_api_client_simple.py`. После запуска открывается окно с элементами управления и областью видеопотока.

## Работа с интерфейсом
В верхней панели задается адрес API и номер камеры. Кнопка запуска камеры начинает захват видео. Кнопка предсказания включает отправку кадров на API и обновление результатов. В области ниже выводятся направление взгляда и значения углов по горизонтали и вертикали, а на сетке показывается точка текущего взгляда.

## Результаты работы
Приложение предоставляет наглядную визуализацию предсказаний модели в реальном времени и предназначено для демонстрации и проверки корректности работы API.
